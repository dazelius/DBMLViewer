# 바이블테이블링 자료형 규칙

## 핵심 규칙
- **데이터 수정 시 컬럼의 자료형(type)을 반드시 확인하고 결과값의 타입을 맞출 것**
- `int` 컬럼에 소수점 값(예: 125.5)이 들어가면 안 됨

## 상세
- `multiply` 액션 사용 시 특히 주의: int 컬럼에 1.2를 곱하면 소수점이 발생할 수 있음
  - 예: int 125 × 1.2 = 150.0 (OK) / int 125 × 1.3 = 162.5 (❌ 소수점 발생!)
- int 컬럼에 곱하기/나누기 적용 시 결과가 정수가 되는지 미리 계산할 것
- 소수점이 발생하는 경우 → `set` 액션으로 반올림한 정수값을 직접 지정하거나, `add`/`subtract`로 정수 증감값을 사용
- show_table_schema로 컬럼 타입을 반드시 사전 확인

## 체크리스트
1. show_table_schema로 대상 컬럼의 type 확인
2. multiply 사용 시 → 현재값 × 배수 = 정수인지 검증
3. 정수가 아니면 → 반올림 후 `set` 액션으로 직접 값 지정
4. float/double 컬럼은 소수점 허용
