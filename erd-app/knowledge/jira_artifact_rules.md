# Jira 일감 아티팩트 표시 규칙

## 1. 링크 필수
- Jira 일감을 아티팩트에 표시할 때 반드시 클릭 가능한 링크를 포함한다.
- 링크 형식: `<a href="https://krafton.atlassian.net/browse/AEGIS-XXXX" target="_blank" class="jira-key">AEGIS-XXXX</a>`

## 2. 하이어라키(에픽/하위작업) 표시
- 에픽(Epic)과 하위 작업(Task/Story)이 있는 경우 반드시 계층 구조로 표시한다.
- 에픽 카드: 강조 스타일 (border-left 색상, 배경 강조, "Epic" 뱃지 표시)
- 하위 작업: 에픽 아래에 들여쓰기 + 세로 연결선으로 하이어라키를 시각적으로 표현
- 각 일감에 이슈 유형(Epic/작업/스토리/버그) 뱃지를 표시
- 레이블이 있으면 함께 표시 (작은 텍스트로)

## 3. 표시 정보
- 이슈 키 (링크)
- 이슈 유형 뱃지
- 제목
- 담당자
- 레이블 (있는 경우)
- 보고자 (에픽에만, 선택)

## 변경 이력
- 2025: 링크 필수 규칙 추가
- 2025: 하이어라키 표시 규칙 추가
