# Jira 댓글 작성 스타일 규칙

## 핵심 규칙
- Jira 댓글 작성 시 **마크다운 서식을 사용하지 않는다**.
- `##`, `###`, `**굵게**`, `---`, `-` 리스트 등 마크다운 기호 금지.
- Jira 웹 UI에서 마크다운이 렌더링되지 않고 원문 그대로 노출되기 때문.

## 올바른 스타일
- 순수 평문(plain text)으로 작성
- 구조화가 필요하면 줄바꿈과 들여쓰기, 번호 매기기(1. 2. 3.)로 구분
- 강조가 필요하면 대문자나 괄호, 화살표(→) 등 텍스트 기호 활용
- 코드나 변수명은 백틱 없이 그냥 텍스트로 표기

## 예시
```
[가설] 재장전 취소 + 클라이언트 예측 탄약 동기화 불일치

1단계: 탄약 0 → 재장전 시작
탄약 소진(0발)으로 IsNeedReload() true → Reload() → TryReload_Internal() 호출
IsReloading = true, canFire = false 설정

2단계: 점프+사격 연타 → 재장전 취소 발생
점프/사격 입력으로 CancelReload() 호출 가능
→ canFire = true, nextFireTime = Time.time으로 즉시 발사 가능 상태 복원
→ 이 시점에서 _reservedAmmoCount는 0으로 리셋되지 않음
```
